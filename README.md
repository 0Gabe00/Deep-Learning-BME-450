# Title
  ADHD Detection from Brain MRI with Deep Learning
## Team Members
  Gabriel Montoya (0Gabe00)
### Project Description
  This project aims to develop a deep learning model to classify brain MRI scans as either ADHD or neurotypical. I plan to use a publicly available dataset such as the ADHD-200 dataset, which contains structural and functional MRI scans from both ADHD and control subjects. The goal of this project is to train a neural network to distinguish between ADHD and normal brain scans. Multiple models will be evaluated. Model performance will be evaluated using metrics such as accuracy and precision on both training and test datasets. The final objective is to compare different models and determine which approach provides the best generalization performance for ADHD classification from MRI data.
